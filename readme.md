`ln -s ~/fonts/Iosevka ~/.local/share/fonts`

### Apple Emoji

Download .ttf : <https://github.com/samuelngs/apple-emoji-ttf/releases>

`ln -s ~/fonts/AppleEmoji ~/.local/share/fonts`
